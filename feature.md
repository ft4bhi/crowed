

# Olex for Cows – Feature List

---

## 1. Marketplace (Home)

* Video-first livestock listings
* Auto-play muted preview videos
* Like (❤️) listings
* Direct call seller
* Location-based browsing
* Quick filter chips
* Advanced filtering (breed, milk, price, pregnancy, etc.)

* Radius-based search
* Multi-language support
* Save favorites
* View seller profile
* Listing detail page with full media gallery
